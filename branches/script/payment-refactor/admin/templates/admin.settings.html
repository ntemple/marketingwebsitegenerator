Password - Your password was created by you when you installed the script or you gave it to your installer for him to install for you. If you have forgotten your password here is what you need to do to retrieve it. Use a FTP Client to connect to the server and folder (/public_html/site_name) where you have the script installed. Go to Folder config. There you will find a file named constants.php. Look for the line that looks like this: define("ADMIN_PASSWORD", "password"); // database password. If you change that password in that file, this will be the new password you need to login in the future.

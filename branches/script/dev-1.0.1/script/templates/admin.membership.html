Create/Edit Custom Memberships

Use this section to Edit an upgrade membership like 'Pro' Or 'Gold'. Deactivate any memberships you are not using. If you ever need a new membership to add, simply re-activate one of the ones you currently do not use. The Memberships are currently ranked from lowest at the top down to highest at the bottom. Use the arrows to move the up/down in rank. At any given active membership level, the member will see options to upgrade to memberships below it that are active and also checked accordingly. The upgrade options active and checked will be listed in order to rank as you have them listed here. Use the check boxes under each individual membership to customize which upgrades should be shown to any specific membership level.

Edit Description Page | Edit Members Download Page | is to be used for the member to see when they click on the "Membership" in their login area. They will see their current status. The Edit Members Download Page allow them to access the bonus related to that membership. You just need to add your download items or information to that page. If they are seeing an upgrade option for that membership, the Edit Description Page is the description of benefits they will see as to why to upgrade. Your payment buttons are automatically on this page for you. When Done, set the prices for these at  "Membership Prices" on the menu above

Referrals Needed, allows you to set a number of referrals they make to the system so that they can get an upgrade to any membership you choose. Leave at 0 to deactivate.

Promotion Codes can be used to gift people to upgraded memberships. They are great to tell a JV Partner to use to email to their list to get an upgraded bonus or used for support issues to satisfy customers. You can choose to only use when people join on the signup page, or only inside the members area. Or you can choose to have the option available in both areas. So you can say, use xxxx when joining. Current members, login and add xxxx promotion code in your profile. If you do not want to use this feature, just select "Deactivate." You can make the promotion code any secret text you like with NO SPACES.

Custom fields added to database:
Use this section to add new fields to the database to be used on signup and other areas of the site. For instance, you may want to ask people what their Date of Birth is, favorite hobby, or type of gold club they use etc.

{error} {field_list}
 
Field Name Action


Field Name:

This is the name of the field that will be added to Database.
Do not use spaces. Use underscore  _  (like bank_name)
     
Field Description:

Description of field
If field is used at signup this will be the text displaying in front of it. i.e. field name= first_name, field description= First Name
     
Field Default Value:

If a member doesn't put anything in the field at signup the field will be filled automatically with this value.

     
Field will contain:
Text (and or Numbers)


Number (Value, may be used in Calculations)
     
Field Length:

Leave blank for default length
(255 characters for text fields, 11 for numbers.
This is to control the size of text saved by members)
     
Tag for field:

Enter the TAG name you want to use in your pages and emails.
(blank for no tag) i.e. favorite_color for field color.
The { } or [[ ]] around the tags will later be shown on each page where you can use this tag. - Do NOT add the { } now.

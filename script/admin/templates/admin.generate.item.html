Generate new site page
Use this section to add new pages to your site. The pages will be made for you to modify in later on the | Site Templates | link on the menu above. It will also generate the .php page needed in the script automatically for you.

The page will also show on your menu if you like. If you do not want it to display, or only want it to display to certain members, just go to | Menu | option on the main admin menu above when finished.

WARNING: Use same php name as old one file (without .ag.php)
Example: if you want to update file "test.ag.php" you should put as Menu item name "test"
If you do not want it to display, or only want it to display to certain members, just go to | Menu | option on the main admin menu above when finished.

Paste the the html for new page. You may use your defined tags.
You may leave blank for now. You can go to | Site Templates | later on the menu to edit this page.

Tags to be used in templates:

{tag_list}

To add payment buttons for your products these below:
 On any page that you want to add a payment button for any product, simply paste this code where you want the order button to appear to a live visitor.

{product_list}

Note: The page will be added to main menu or the member area menu if it is a member area page. The page will also show on your menu if you like. If you do not want it to display, or only want it to display to certain members, just go to | Menu | option on the main admin menu above when finished.

Note: Please make sure that you have CHMOD 777 the site root directory and the templates directory, for the script to be able to write the files on local file system.
